## Windows 기능 켜기

제어판 > 프로그램 > 프로그램 및 기능 > windows 기능 켜기/끄기 > ‘Linux용 windows 하위시스템’과 ‘가상 머신 플랫폼’ 체크 후 확인 > 컴퓨터 재시작

<br />

## WSL2 설치하기

- 설치하기

```html
wsl.exe --install
```

<p></p>

- WSL 2 기본 버전 설정

```html
wsl --set-default-version 2
```

<p></p>

- WSL 2 버전 확인

```html
wsl --list --verbose
```
